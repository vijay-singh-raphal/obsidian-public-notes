## <span style="color:#ED6AFF;">What is a data structure?</span>
A **Data Structure** is a way of organizing and storing data so that it can be used efficiently.
#### <span style="color:yellow;">Definition</span>
A data structure is a collection of data values and the relationships/operations that can be applied to that data.
<span style="color:orange;">Example Array</span>
```python
[1, 5, 6, 8]
['a', 'z', '3']
```
These are collections of data values stored in sequence.
## <span style="color:#ED6AFF;">Why Do We Need to Learn Data Structures?</span>
To write efficient code and crack coding interviews.
<span style="color:orange;">Problem Solving Flow</span>
Input → Algorithm → Output
Data structures help in managing data properly while designing algorithms.
## <span style="color:#ED6AFF;">Why Different Data Structures Exist</span>
A problem can often be solved in **multiple ways**.
But some data structures are:
- Faster
- More memory efficient
- Better suited for certain operations
Means take less time or space to solve the problem.
