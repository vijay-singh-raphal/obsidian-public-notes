## Font Group
<span style="color: yellow; font-size:25px">Font Related work</span>
![font group | 500](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEghNkgZ0PojN_yMXMLruh0GAyjSmZNIig9N8lE6JEA6f2hhK2xT1q3VW9TGrVhwnwlvyO_DdVzRap0d5F1saMjMaYo5R3YFIUfH2stcrCWzh3r3sRMqXTm9kl4z9pcGRWf5EWkGsnnqtapzKxpKua3ukgifkGF2KKYqFGeqrM1CAKZH1mtJlsCLgRY5ceOi/s1600/MS-Excel%20Font%20Group.jpg)
Font Group = Tools to make your text look better
Main options in Font Group:
- Bold (B) → Makes text dark
- Italic (I) → Slants the text
- Underline (U) → Draws a line under text
- Font Style → Change text design (Arial, Calibri, etc.)
- Font Size → Increase or decrease size
- Font Color → Change text color
- Fill Color → Change background color of cell
- Borders → Add lines around cells
## Clipboard Group
<span style="color: yellow; font-size:25px">Clipboard Related work</span>
![clipboard | 300](https://i0.wp.com/excellopedia.com/wp-content/uploads/2024/04/Accessing-the-Clipboard-Panel.jpg?fit=252%2C208&ssl=1)
Clipboard Group in Excel is used to copy, cut, and paste data from one place to another.
Main options in Clipboard Group:
- Cut (✂️) → Moves data from one place to another
- Copy (📄) → Makes a duplicate of data
- Paste (📋) → Places copied or cut data
- Format Painter (🖌️) → Copies formatting (style) from one cell to another
## Alignment group
<span style="color: yellow; font-size:25px">Alignment Related work</span>
![alignment | 400](https://www.exceldemy.com/wp-content/uploads/2022/10/1-Types-of-alignment-in-Excel-1.png)
Alignment Group in Excel is used to arrange and position text inside a cell.
Main options in Alignment Group:
- Left Align → Text moves to left
- Center Align → Text stays in middle
- Right Align → Text moves to right
- Top / Middle / Bottom Align → Adjusts vertical position
- Wrap Text → Shows long text in multiple lines in same cell
- Merge & Center → Joins multiple cells and centers text
## Number Group
<span style="color: yellow; font-size:25px">Number Related work</span>
![number | 600](https://www.learningcomputer.com/images/excel2007/excel_home_tab4a.jpg)
Number Group in Excel is used to format numbers in different styles like currency, percentage, dates, etc.
Main options in Number Group:
- General → Default format
- Number → Simple numbers with decimals
- Currency (₹) → Shows money values
- Percentage (%) → Converts number into percent
- Comma ( , ) → Adds commas (e.g., 1,000)
- Increase Decimal → Adds more decimal places
- Decrease Decimal → Reduces decimal places
#### Currency and Accounting
| Feature         | Currency 💰            | Accounting 🧾              |
| --------------- | ---------------------- | -------------------------- |
| Symbol position | Close to number (₹500) | Far left aligned (₹ 500)   |
| Zero value      | Shows ₹0               | Shows dash (–)             |
## Autofill
<span style="color: yellow; font-size:25px">How autofill work</span>
![fill | 400](https://www.wallstreetmojo.com/wp-content/uploads/2018/12/AutoFill-in-Excel-Example-1-Fill-Flash-option.png)
AutoFill in Excel is used to automatically fill data in cells without typing again and again.
Examples:
- 1, 2 → drag → 3, 4, 5…
- Monday → drag → Tuesday, Wednesday…
- Jan → Feb → Mar…
- What AutoFill can do:
Continue number series
- Fill days, months
- Copy formulas
- Repeat same value
## Basic Function
<span style="color: yellow; font-size:25px">Working on basic function</span>
![basic function | 500](https://static.wixstatic.com/media/2d24e3_b9e65b99ee974881a90765a74070f8fb~mv2.png/v1/fill/w_638,h_228,al_c,q_85,enc_avif,quality_auto/2d24e3_b9e65b99ee974881a90765a74070f8fb~mv2.png)
Basic functions/ functions in Excel are pre-defined formulas used to perform calculations easily.
Basic Functions
- SUM() → Add numbers
- AVERAGE() → Find average
- MIN() → Smallest value
- MAX() → Largest value
- COUNT() → Count numeric cells
- COUNTA() → Count non-empty cells
